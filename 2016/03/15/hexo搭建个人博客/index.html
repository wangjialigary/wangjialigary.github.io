Hexo + Github搭建静态博客

  • Github
  • Hexo
前端

之前一直想自己搭个个人博客，又觉得搭博客应该是很复杂的是，自己能力不够，
所以一直没动手做，直到最近不做不行了，于是到网上查查发现好像也不太难，按步骤下来其实也就几个小时的事，只不过美化完善需要费点时间，很多时候我们都是被自己的想象吓跑了
言归正传

一，环境

  1. 安装Git，node.js
  2. 配置Github
    首先登陆github，创建一个repository，名字必须为xxx.github.io，xxx为自己的账户名
  3. 安装Hexo
    npm i -g hexo 安装完成后查看
    hexo -v 如果出现版本信息表示安装成功

    二，创建博客

  4. 在电脑中建立一个名字叫“Hexo”的文件夹，然后在此文件夹中右键打开Git
    Bush
    执行命令$ hexo init 进行初始化，hexo随后会自动在目标文件夹建立网站所需要的文件。
  5. 运行$ npm install 安装生成器
  6. 本地运行hexo，打开浏览器，输入localhost：4000，就可以在本地看到自己的博客了，Ctrl+C停止运行

    三，发布到Github

  7. 配置根目录下的_config.yml，打开它拉到最后一行，把Github配置信息填上去

  8. 输入hexo d 按照提示输入Github的登录名和密码，在浏览器输入网址：xxx.github.io就可以访问自己的博客啦

    四，更换主题，hexo主题很多，选一个自己喜欢的执行两步就可以了

  9. 把主题clone到themes下
  10. 修改根目录下theme

    五，发布文章

    控制台输入$ hexo new “文章名”，生成一个md文件，用编辑器打开，编写完后保存，
    输入$ hexo d -g 发布更新博客，发布时按要求输入Github账号密码，发布成功后，访问xxx.github.io就可以看到啦
