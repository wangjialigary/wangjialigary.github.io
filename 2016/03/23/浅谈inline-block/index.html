浅谈inline-block

  • css
  • inline-block

display:inline-block在水平和垂直方向会产生空白间隙，这是为什么呢？主要是空白符的存在导致的，换行符，回车符，空格等都属于空白符，这些都会导致间隙的出现，那么要怎么去除间隙呢？

一、改变HTML结构

元素间留白间距出现的原因就是标签段之间的空格，因此，去掉HTML中的空格，自然间距就木有了。可以这样写：

甚至不考虑可读性，可以把代码都写在一行

二、负的margin

div{
    font-size: 12px;
}
div a{
    margin-right: -4px;
    *margin-right: 0;
}

这种解决方法并不完美，如果父元素的字号不一样，可能-4px就不能解决问题。

三、设置父元素字体为0

设置父元素字体为0，然后在“inline-block”元素上重置字体需要的大小，同时设置vertical-align: top。
div{
font-size: 0;
}
div a{
vertical-align: top;
display: inline-block;
display: inline; zoom: 1;
}
